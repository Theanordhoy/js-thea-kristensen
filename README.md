# Rainydays JavaScript 1

![Landing page](/assets/rainydays-js-index.png)

## Description

This project was a course assignment in JavaScript 1. Rainydays is a responsive e-commerce website showing products from an API. You can browse the different products, sort products by gender and see detailed information about each product. You can add the products to cart and change the quantity on the checkout page. You can also remove products and clear the cart. 

## Built With

- HTML
- CSS
- JavaScript

## Getting Started

### Installing

1. Clone the repository

```bash
git clone https://github.com/Theanordhoy/js-thea-kristensen.git
```

2. Navigate to the project folder

```bash
cd js-thea-kristensen
```

3. Open the project in code edit(VS code)

```bash
code .
```

4. Start the project using Live Server

- Open index.html using Live Server Extension
